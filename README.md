# Technologies Used

HTML
CSS
SwiperJS

# Features
1. Developed following the mobile-first methodology.
2. Uses media queries to adapt to different screen sizes.
3. Includes a hamburger menu for mobile devices.
4. Incorporates a carousel using SwiperJS.
